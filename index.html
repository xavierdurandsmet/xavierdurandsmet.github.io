Parks of San Francisco

"Parks of San Francisco" is a Web Application, which enables the user to find all sorts of of parks in San Francisco, by sorting public city data. The user can find a type of park on the map, and create an itinerary between two parks.

Technologies used:

JQuery, Ajax, MEAN Stack, LeafletJS

CodeCast

CodeCast is an interactive screencasting application for programming instructors and students. Teachers can initiate coding sessions to livestream a working REPL, and users can play back any previous session at their convenience. At any point during a live or recorded lecture, students can fork the displayed text, allowing them to modify, run, and save their own variations on the instructor's code, without interfering with the main code!

Technologies used:

MEAN Stack, SocketIO

Dragg'z

Dragg'z is a Chrome extension that gives users an easy way to modify the structure of web pages by dragging and dropping DOM elements. Changes are persisted to a remote database enabling users to share their modifications.

Technologies used:

MEAN Stack, Chrome Extension, JQuery

Reminiz Bot

Reminiz Bot is a Messenger ChatBot used by Face Recognition Start-up Reminiz at the SXSW Event in 2017. Reminiz Bot enables the user to know who is live on TV and get information on the person.

Technologies used:

NodeJS, Facebook APIs, Reminiz APIs
